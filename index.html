How to download wallpapers?

NOTE: If you want to download wallpapers, you need to have Google Drive account.

Follow these steps to download our wallpapers:

On a laptop / PC:

Step 1: Click "Download via Google Drive"

First, check if the button has line going through it. If it does, that means it's impossible to download the wallpaper by now, so don't click it (if you do so, nothing will happen). But if it doesn't, that means downloading the wallpaper is avaliable, so if you click that'll open a Google Drive tab.


Step 2: Download the .zip file

In a Google Drive tab, you'll should see 3 icons near the top-right corner: Google Drive icon with "+", Download icon and 3 dots. You should click the middle icon (which is download icon), and if you do so, there are 2 things that can happen:

1: The .zip file will start downloading (which, obviously, SHOLUD happen)
2: It can warn you about the large size of the .zip file (which usually happens with the "high quality" wallpapers), so you click the button that says "Download".

If you're on Google Chrome, you'll find the .zip file downloading on bottom-left corner. If you're on other browser, like Microsoft Edge, Mozilla Firefox, Opera or other ones, you'll see the .zip file downloading in "Download" tab.


Step 3: Extract the .zip file

Try to find the .zip file in File Explorer if you're Windows, in Files if you're on Linux, and Finder if you're on Mac. If you've found the .zip file, right-click it. If you've found the "Extract" button, click it and extract it.

Step 4: Set as Background

Open the extracted folder, select the wallpaper, right-click it, and click "Set as background". You can now enjoy our wallpaper collection :)

On a mobile device:

Step 1: Click "Download via Google Drive"

First, check if the button has line going through it. If it does, that means it's impossible to download the wallpaper by now, so don't press it (if you do so, nothing will happen). But if it doesn't, that means downloading the wallpaper is avaliable, so if you press that'll open a Google Drive tab.


Step 2: Download the .zip file

In a Google Drive tab, you'll should 3 see icons near the top-right corner: Google Drive icon with "+", Download icon and 3 dots. You should click the middle icon (which is download icon), and if you do so, there are 2 things that can happen:

1: The .zip file will start downloading (which, obviously, SHOLUD happen)
2: It can warn you about the large size of the .zip file (which usually happens with the "high quality" wallpapers), so you press the button that says "Download".

You'll see the .zip file downloading in "Download" tab.


Step 3: Extract the .zip file

Try to find the .zip file in Files > Downloads. If you've found the .zip file, press and hold it. If you've found the "Extract" button, press it and extract it. Open the extracted folder, press and hold the wallpaper, then press "Share", and select "Photos" on iOS / iPadOS or select "Gallery" on Android.

Step 4: Set as Background

Go to Settings, then Wallpapers, create a new one, and select the wallpaper. You can now enjoy our wallpaper collection :)
